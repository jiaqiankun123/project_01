git 分4个工作区域: 1. 工作目录 (Working Directory) 2. 暂存区 (Stage/index) 3. 资源库/本地仓库 (Repository/ Git Directory) 4. 远程仓库(云端仓库) (Remote Directory) git add . 将代码提交到缓存区 git commit -m"" 给本次代码提交添加注释备注 git push 将代码推送到远程仓库 git clone 拉去代码 git pull 代码的拉取（代码的同步） git init 初始化本地仓库 git rm -r -f -cached 删除缓存 git branch 查看分支 git branch dev 创建分支dev git checkout dev 切换至dev分支 git merge dev 把分支dev合并到当前分支 git push origin dev 将dev分支推送到远程仓库 git checkout master 切换至master分支 git push origin -d dev 删除分支 DOS指令 查看当前路径: pwd DOS切换盘符: D: 查看当前路径下的目录： dir/ls 进入到单级目录： cd 文件夹名字 创建文件夹： mkdir aa 退回到上一级目录： cd .. 进入到多级目录： cd 文件夹名1\文件夹名2 退回到盘符目录: cd \ 清屏 : cls 退出命令: exit DOS 快捷键 上箭头 ⬆ : 上一条命令 tab 键 : 路径补全 esc 键 : 清空当前命令
